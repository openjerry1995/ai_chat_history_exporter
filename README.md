# AI Chat History Exporter

Save your AI conversations permanently. Export single chats or your entire history from Grok, Gemini, ChatGPT, and Claude — all in clean Markdown format.

[![Chrome Web Store](https://img.shields.io/badge/Chrome-v2.0.0-blue?style=flat-square)](https://chrome.google.com/webstore)

## Install

Search **"AI Chat History Exporter"** in the [Chrome Web Store](https://chrome.google.com/webstore) and click Add to Chrome. That's it — no login, no setup.

## Usage

1. Open a chat tab (ChatGPT, Claude, Grok, or Gemini)
2. Click the extension icon in your browser toolbar
3. Choose **Export Current Chat** to save the conversation you're looking at, or **Export All History** to download your entire conversation history into one file
4. Your file downloads automatically as Markdown

## Supported Platforms

| Platform | Current Chat | All History |

## Privacy

The Extension runs **100% locally**. No data is collected, stored, or sent to any server. Your conversations never leave your computer.
